#### Generative Adversarial Networks (GANs) for Credit Card Fraud Data

View this notebook on nbviewer: https://nbviewer.jupyter.org/github/codyznash/GANs_for_Credit_Card_Data/blob/master/GAN_comparisons.ipynb
